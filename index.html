DuckHunt2020:

An Augmented Adeventure

For my AR/VR Specialization portfolio project I intended to create a new, modern version of the 1984 Nintendo classic Duck Hunt.

This game was designed to be more immersive and interactive than any other game on the market at the time. And it is a game, that in my opinion, was far ahead of its time. However Duck Hunt has never gained the same success in later releases, as other titles in the Nintendo family.

So I built a 21st Century version, that is relevant with today's available technology.
